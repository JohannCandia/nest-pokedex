# EJECUTAR EN DESARROLLO

1. Clonar el repositorio 
2. Ejecutar el siguiente comando:

```
yarn install
 ```

 3. Tener instalado NEST CLI 

 ```
 npm  i -g @nestjs/cli
 ```

 4. Levantar la base de datos

 ```
 docker-compose up -d
 ```


##  Stack Usado
* MongoDB
* Nest
